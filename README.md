
# ReactFlow Mini PoC

Локальный минимальный проект на React + ReactFlow + Zustand.

https://react-flow-poc-six.vercel.app/

## Запуск

```bash
# 1) Установите зависимости
npm i

# 2) Запустите дев-сервер
npm run dev
# Откройте http://localhost:5173

# Опционально: сборка и предпросмотр
npm run build
npm run preview
```
